dat.GUI demo

Score: 0

dat.GUI is a JS library for creating a minimalist GUI - go ahead an click the "Open Controls" button on the right.

This allows your users to focus on the experience you have created, rather than being forced to scroll up and down the screen while they are tweaking the parameters of your app.

We are using ES6 module syntax here - so this example needs to run off of a web server.

Read the source code to see how it works.
